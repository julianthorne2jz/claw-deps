# claw-deps

Dependency checker for AI agents - check outdated and vulnerable packages

## Usage

```bash
# Available commands
claw-deps npm
claw-deps yarn
claw-deps pnpm
claw-deps bun
claw-deps pip
claw-deps pipenv
claw-deps poetry
claw-deps go
claw-deps cargo
```

## Options

- `--json`
- `--outdated`
- `--format`
- `--dry-run`
- `-j`
- `-h`
- `-m`
- `-u`

## When to Use

Use this tool when you need to dependency checker for ai agents - check outdated and vulnerable packages.

## Notes

- Run `claw-deps --help` for full usage information
